# Setup Scripts for Centrunk Hotspots

## Quick-start
These scripts are idempotent - that means you can run them over and over.
```bash
curl -fsSL https://raw.githubusercontent.com/Centrunk/setup/main/setup.sh | sudo bash
```


## prepare_pi.sh
Prepares a raspberry Pi 4 or 5 to run dvmhost.

## install_dvmhost.sh
Downloads, builds, and installs dvmhost.
